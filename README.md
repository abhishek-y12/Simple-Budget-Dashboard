# Simple-Budget-Dashboard
Developed an interactive budget tracking dashboard in Excel sheet to manage monthly finances, analyze categorized expenses, and visualize spending trends using automated calculations and charts.
Monthly Budget Dashboard (Excel Sheets)

Overview
This project is a dynamic monthly budget dashboard built using Excel Sheets to track income, manage categorized expenses, and monitor overall cash flow. It uses automated formulas and charts to provide a clear view of planned versus actual spending.

Features
Automated expense aggregation using SUMIF and SUM formulas
Cash flow calculation to track remaining balance
Budget vs Actual comparison through charts
Category-wise spending breakdown with percentage analysis
Structured transaction tracker for real-time financial updates

Tools Used
Excel Sheets, Data Analysis, Financial Modelling, Data Visualization, Spreadsheet Automation

How It Works
Users enter transaction data in the expense tracker, and the dashboard automatically updates category totals, cash flow, and visual insights to support better financial planning.
